# Fav Logo: `FxFavIcon`

## Usage

```tsx
import { FxFavIcon } from '@fluctux/ui'

<FxFavIcon variant='dark' className='invert-fav' />
```

## Props

| Prop Name  | Type                                | Description                                                                                                                                         | Default Value  |
|------------|-------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|----------------|
| `size`     | `keyof typeof iconSizeVariants`     | Defines the size of the icon. Available options: `sm` (20px), `md` (30px), `lg` (50px), `xl` (80px).                                                 | `md` (30px)    |
| `variant`  | `keyof typeof iconVariants`         | Defines the color variant of the icon. Available options: `light`, `dark`, `default`.                                                                | `light`        |
| `className`| `string`                            | A custom class name for additional styling of the icon.                                                                                             | `""` (No custom class)  |
